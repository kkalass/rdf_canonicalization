Creates a canonical wrapper around the given graph

The provided graph is wrapped (not copied) and used for all graph operations. The canonical form will be computed lazily on first access to canonicalNQuads, hashCode, or operator ==.

Parameters:

  • graph The RDF graph to wrap with canonical equality semantics
